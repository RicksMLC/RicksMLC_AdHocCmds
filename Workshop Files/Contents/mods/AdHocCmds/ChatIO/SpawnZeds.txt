-- SpawnZeds.txt - ChatIO file for spawning zombies.
--
-- More than one zombie spawn file can be used in the chatInput.txt file.  To make a valid spawn file, the 
-- "type=spawn" line must be included.
--
-- The zombies can be targeted to any player on a server by setting the "player=" to the username of the player.
-- For example, to target me as a player the attribute would be "player=RicksMLC".  In single player play this 
-- attribute is ignored.
--
-- If the username cannot be found the zombies are spawned relative the host player.
-- Default is "player=" with no value set, so the host is targetted.
--
-- Mandatory attributes are:
--      "zedname=<Name for Dogtag>" Can be alpha-numeric and include some punctuation but must 
--                  not include "," "(" ")" ":"
--      "radius="   Spawn the zombies within a circle of this radius.  If not set the default is 10.
--      "offset="   Offset to move the zombie spawn point from the player, (Ie: spawn at least this distance
--                  from the player) (Default is 12)
--      "facing="   Set to any value (eg: "yes") so the zombies will spawn "offset=" distance directly in front
--                  of the player. If this is set the "radius=" is ignored.
--      "zCount1=n" Number of zombies to spawn with the given gender and outfit.
--      "gender1="  Gender of zombies to spawn.  Leave blank to chose random or outfit dependant 
--                  gender (see note below for details).
--      "outfit1="  Outfit the zombies will have when spawned.  This can override the "gender="
--                  setting (see note below for details).
--      "line1="    Radio broadcast lines.  These lines will appear on the radio if the radio is on and tuned
--                  to Radio C.H.A.T (108.6MHz)

-- Note: The "outfit" will override the "gender".  If the specified gender does not match the outfit's required
-- gender then the gender for the outfit will be used instead of the specified gender.  If the same outfit is 
-- available for both genders (eg "chef") and "gender=" is blank then it will generate a randomised group of 
-- male and female zombies in their respective versions of the outfit.

-- More than one set of zombies can be generated by appending the next sequence number to the "zCount", "gender"
-- and "outfit" attributes. It is required that all three of these attributes be present for each set, even if 
-- they are not used.  For "zCount=n" n must be > 0 and is mandatory, and is the number of zombies of the corresponding 
-- outfit. If "gender" or "outfit" have no value assigned a random one will be selected.
--
-- The default example below will generate one male office worker, one female office worker (in a skirt), and 
-- two random zombies.

type=spawn
player=

zedname=Harold P. O'Qwerty_01-Jr
radius=
offset=10
facing=yes

spawnPointPreference=SameBuilding
minArea=2

zCount1=1
gender1=m
outfit1=OfficeWorker
zCount2=1
gender2=f
outfit2=OfficeWorkerSkirt
zCount3=2
gender3=
outfit3=

line1=Warning: There are reports of more erratic individuals, who are to be considered dangerous.
line2=These individuals are identifiable by their dog tags.
